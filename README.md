# 🎮 Chat Game Bot - Игровой бот для Telegram чатов

Бот для развлечения в групповых чатах Telegram с системой роста, рейтингами и настраиваемыми приветствиями.

## 🚀 Возможности

### 🎯 Основные команды
- **`/up`** - Увеличить свой размер (от 0.1 до 5 см, раз в 5 минут)
- **`/size`** - Посмотреть свой текущий размер
- **`/top`** - Топ-10 самых больших размеров в чате
- **`/help`** - Показать справку по командам

### 👑 Админ-команды (только для администраторов чата)
- **`/set_welcome [текст]`** - Установить кастомное приветствие
- **`/reset_welcome`** - Сбросить приветствие до стандартного
- **`/set_goodbye [текст]`** - Установить кастомное прощание
- **`/reset_goodbye`** - Сбросить прощание до стандартного

### 🎨 Особенности
- 🎯 Случайное увеличение размера с разными уровнями редкости
- ⏱️ Защита от спама (раз в 5 минут)
- 🏆 Рейтинговая система с медалями
- 👋 Автоматические приветствия новых участников
- 😢 Прощание с уходящими участниками
- 📝 Полная кастомизация приветствий/прощаний
- 💾 Локальная база данных SQLite
- 🔒 Работает только в группах (в ЛС предлагает добавить в чат)
